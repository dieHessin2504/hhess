# SEO-Konventionen (projektspezifisch)

## Meta-Angaben
- **Meta-Title: max. 60 Zeichen** (sonst kürzt Google in den Suchergebnissen ab). Genau eine `<title>` pro Seite, aussagekräftig, wichtigstes Keyword vorn.
- **Meta-Description:** ~150–160 Zeichen, mit Nutzen/CTA; pro Seite einzigartig.

## Indexierung
- Jede Seite bekommt ein explizites `<meta name="robots" content="index, follow">` (kein Verlass auf den impliziten Crawler-Standard). Vor dem Setzen von Robots-Tag, Title und Description wird der User dazu **immer aktiv gefragt** (mit eigenen Vorschlägen für Title/Description) — nicht selbstständig festlegen.
- Stand 14.07.2026: `index.html`, `kundenstimmen.html`, `youtube.html` haben `index, follow`.
